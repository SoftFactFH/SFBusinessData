====================================================================================================================
						README SFBusinessData
====================================================================================================================

SFBusinessData sind Komponenten für Embarcadero Delphi zur wiederverwendebaren, datenbankunabhängigen Programmierung
von Business- bzw. Datenlogik.

1. Informationen und Beispiele

Informationen und Beispiele zur Verwendung von SFBusinessData finden Sie:
- in der Dokumentation im Unterverzeichnis \docs
- im beigefügten Demoprojekt (s. \demo)
- in den Einführungsvideos bei YouTube - s. https://www.youtube.com/channel/UCWj1u5gm0Sdj_dX64jyCx9g

2. Erstellungen eigener Businessklassen

Zur Erstellung eigener Businessklassen können Sie die Vorlage \template\UBusinessDataTmpl.pas verwenden
